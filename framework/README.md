
## How to use framework 

### Switch data plane

Complete the interface.p4 file under ./user/switch_data_plane/interface/, following the comments. Files included by interface.p4 can also exit under the same directory.

### Backend server

Implement the application under ./user/server_lib/interface/.

### Generate executable files

Configurate config.json and run generate_w_backbone.py under each directory to generate executable files.
